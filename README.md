# 🤖 AI Resume Job Matcher

An AI-based system that analyzes resumes and compares them with job descriptions to generate match scores, skill gaps, and hiring recommendations.

---

## 🚀 Features

* Resume analysis
* Job description matching
* Skill gap detection
* Match score generation
* Hiring recommendation system
* LLM-ready architecture (future integration)

---

## 🧠 Tech Stack

* Python
* AI / Machine Learning Concepts
* Rule-based Matching System
* LLM-ready design (future scope)

## ▶️ How to Run

1. Clone the repository
2. Open project in VS Code
3. Run the following command:

```
python app.py
```

4. Paste your resume
5. Paste job description
6. Get AI analysis result
